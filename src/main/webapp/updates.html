Changelog

Here I will present you with the most recent updates to PA Stats :)
Note that you can find the latest issues on Github: LINK

v14.1.1

  • The units in the Army composition display now link to the PA Matches Wiki.

v14.1.0

  • Fixed APM Charts
  • Fixed lots of things about the army composition display. It should no longer lag any browser
  • Changed army composition layout a bit to prevent lots of scrolling
  • The units in the army composition can now be click to open pa-db.com. This will hopefully soon be replaced with links to the pamatches.com wiki, which is better maintained
  • Added PA Stats options to the PA-settings

v14.0.0

  • Added capturing of unit constructed and destroyed events
  • Show army composition of players based on those events

v13.2.0

  • Added link to the -under construction- ladder page
  • Fixed an internal bug

v13.1.0

  • Fixed a bug in the mod that caused the ingame-checkboxes to appear next to the controls for Planet smashing
  • Added an ingame-explanation of what PA Stats does, so everyone knows their data is being collected
  • Improved the seachbox for the player-search
  • Link Github issues in the changelog
  • changed a few colors and made the vs between teams white so it is easier to see
  • added a graph on the player-pages. More features will be added to this in a future version
  • Fixed a few internal bugs that probably went unnoticed so far

v13.0.1

  • Fixed a bug that prevented new users from using PA Stats. Sorry. :(
  • Added Updates section

v13.0.0

  • PA Stats is now on Github, feel free to contribute: HERE I am especially looking for people with webdesign knowledge
  • Teamcolors are now used in the graphs. The redundant legend was removed.
  • Players are grouped by teams.
  • The backend has been rewritten for the most part. If you spot new bugs please report them.
  • Old data still uses the old colors, however for old teamgames the list of players is a bit broken. The old data is just too... well it doesn't know of teams.
